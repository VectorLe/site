Lindy Effect

Applied to psychology, we know that "while our knowledge of physics has not been available to the ancients, human nature was".[10] Human nature in this case can be considered robust and the Lindy effect can thus be used as a filter for determining which human behaviors people are most likely to continue to engage in. This is in opposition to looking at academic social science research for insights on human nature, considering the current replication crisis.

Programs I Use

Here are some programs I like to use to get things done efficiently.

  • Web Browser: Firefox. I use Firefox with the following add-ons:
    1. Ghostery
    2. HTTPS Everywhere
    3. Stylish. Dark theme everything. Go to here to get skins.
    4. uBlock Origin
    5. Vim Vixen. Vim Vixen allows the user to nagivate through the web using vim bindings, keeping the user's hands on the keyboard at all times.
  • Window Manager: i3-gaps. i3-gaps is a fork of i3wm which is a tiling window manager that maximizes screen efficiency and allows for multiple workspaces and customizable hotkeys.
  • Search Engine: DuckDuckGo. DuckDuckGo is a search engine that supports user privacy and avoids providing different users different search results based on personalization. It also supports its own !Bang search shortcuts for quickly nagivating to different websites.
  • Email: Protonmail. Protonmail is a secure email service with built-in end-to- end encryption and does not require users to provide sensitive personal information to register.
  • Mobile Payments: Cash App. Cash App has a clean, intuitive interface that allows for simple transactions between friends and also allows users to buy, sell, and withdraw Bitcoin.
